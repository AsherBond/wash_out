ABOUT
WashOut is a gem that greatly simplifies creation of SOAP service providers inside a Rails project.

Rails > 3.0 only. Most of ruby versions will work.
HOW IT WORKS
A SOAP endpoint in WashOut is simply a Rails controller which includes the module WashOut::SOAP. Each SOAP action corresponds to a certain controller method; this mapping, as well as the argument definition, is defined by soap_action method. Check the method documentation for complete info or README below for some live examples.
